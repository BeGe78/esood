Overview

Handles Role model used by devise and cancancan gems.
Only accessible to users with admin role. The routes of this controller support localization but the web pages are not translated as it is used only by admin.

def show
  if @role.users.length == 0 
    @associated_users = "None"
  else
    @associated_users = @role.users.map(&:name).join(", ")
  end
end
